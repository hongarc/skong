# Research Report Format

```markdown
# Research: [Topic]
Date: YYYY-MM-DD

## TL;DR
1-3 sentences: what we found, what we recommend.

## Context
Why this research was needed. What decision it informs.

## Findings

### Option A: [Name]
- **What**: Brief description
- **Pros**: Bullet list
- **Cons**: Bullet list
- **Fit**: How well it matches our needs (Good/Partial/Poor)

### Option B: [Name]
(same structure)

## Comparison Table
| Criteria | Option A | Option B |
|----------|----------|----------|
| Criteria 1 | ✓ | ✗ |
| ...      |          |          |

## Recommendation
"We recommend [X] because [Y]. Main risk is [Z]."

## Sources
- [Source 1](url) — accessed YYYY-MM-DD
- [Source 2](url) — accessed YYYY-MM-DD

## Open Questions
- Things we couldn't determine
- Questions for team/lead
```
