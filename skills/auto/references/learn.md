# Learn & Improve Skills

**MANDATORY after every workflow.**

## Step 1: Reflect
Review what happened:
- New patterns discovered
- Gotchas or edge cases encountered
- Better approaches found
- Missing checklist items
- Unnecessary workflow steps
- New consistency rules

## Step 2: Summarize
```
## Skill Improvements Found

### [skill-name] — [file to update]
- Current: [what exists or "missing"]
- Suggested: [what to add/change]
- Why: [what happened that taught us this]
```

If nothing found: "No improvements identified."

## Step 3: Confirm
Ask: "Found skill improvements. Approve? (all / pick / skip)"

## Step 4: Apply
- all → update all files
- pick → ask which numbers
- skip → done

## Rules
- Never update without confirmation
- Keep changes minimal
- Don't bloat checklists
- Each reference file stays under 100 lines
