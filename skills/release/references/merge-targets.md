# Propagate master to merge targets

For each target, decide CLEAN vs CONFLICT, then act.

## Mergeability check

```bash
# Returns the merged tree if clean; nonzero exit + "<<<<<<<" markers if conflict.
OUT=$(git merge-tree --write-tree origin/$TARGET origin/master 2>&1) || true
if echo "$OUT" | grep -q '^<<<<<<<\|CONFLICT'; then
  STATE="CONFLICT"
else
  STATE="CLEAN"
fi
```

## CLEAN path

Open a PR directly from `master` → `$TARGET`:

```bash
SHORT=$(short_name "$TARGET")           # see mapping below
gh pr list --repo Everfit-io/everfit-api --base "$TARGET" --head master --state open --json number \
  | grep -q '"number"' && { echo "PR already open"; exit 0; }

gh pr create \
  --repo Everfit-io/everfit-api \
  --base "$TARGET" \
  --head master \
  --title "chore(merge): master → $TARGET (v$VERSION)" \
  --body  "Auto-generated by /release. No conflicts detected.\n\nRelease: v$VERSION"
```

## CONFLICT path

```bash
SHORT=$(short_name "$TARGET")
BRANCH="merge_master_to_$SHORT"

# Wipe local + remote stale branches
git branch -D "$BRANCH" 2>/dev/null || true
git ls-remote --heads origin "$BRANCH" | grep -q . \
  && git push origin ":refs/heads/$BRANCH"

git checkout -B "$BRANCH" "origin/$TARGET"
git merge origin/master --no-ff --no-commit || true   # we expect conflicts
```

Now invoke conflict resolution (`references/conflict-resolution.md`). After it returns:

- If any files still need manual review → STOP, print list, ask user.
- Otherwise:

```bash
git add -A
git commit -m "chore(merge): master into $TARGET for v$VERSION"
git push -u origin "$BRANCH"

gh pr create \
  --repo Everfit-io/everfit-api \
  --base "$TARGET" \
  --head "$BRANCH" \
  --title "chore(merge): master → $TARGET (v$VERSION)" \
  --body  "$(cat <<EOF
Auto-merge branch from /release pipeline for v$VERSION.

## Conflict report
- Auto-resolved: <list>
- Manual: <none — or list>

Verify before merging.
EOF
)"
```

## Short-name mapping

| Target | Short |
|---|---|
| develop | dev |
| releasing_staging | staging |
| master-debugging | debug |

```bash
short_name() {
  case "$1" in
    develop) echo dev ;;
    releasing_staging) echo staging ;;
    master-debugging) echo debug ;;
    *) echo "$1" ;;
  esac
}
```
